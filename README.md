# Active Directory Implementation Guide

A comprehensive GitHub Pages website covering Active Directory from concept to production deployment.

## Topics Covered

- **What is Active Directory** — History, components, core concepts
- **AD Service Types** — AD DS, Azure AD / Entra ID, AD CS, AD FS, AD LDS, AD RMS
- **Windows Server Editions** — Essentials, Standard, Datacenter comparison
- **Implementation Solution** — Step-by-step deployment with PowerShell scripts
- **Architecture Design** — Small, medium, enterprise, and hybrid reference architectures
- **Cost Estimator** — Interactive calculator for licensing, hardware, and services
- **Best Practices** — Security hardening, operations, backup, monitoring
- **FAQ** — Common questions, port reference, AGDLP guide

## Live Site

Deploy via GitHub Pages: `Settings → Pages → Branch: main → / (root)`
